#  Sistema de Joyería - MVC + BD + POO

Proyecto desarrollado en Java usando Maven, aplicando el patrón MVC, conexión a base de datos y programación orientada a objetos.

---

##  Tecnologías utilizadas

- Java
- Maven
- MySQL
- JDBC
- Git

---

##  Arquitectura

El proyecto está estructurado bajo el patrón **MVC**:

- **Modelo**: Clases (Tercero, Producto, Venta, DetalleVenta)
- **Vista**: Interfaz gráfica (Swing)
- **Controlador**: Manejo de eventos y lógica
- **DAO**: Acceso a base de datos
- **Conexión**: Clase para conectar a MySQL

---

##  Base de Datos

Base de datos: `joyeria`

Tablas principales:

- tercero
- tipo_tercero
- producto
- venta
- detalle_venta

---

##  Funcionalidades

✔ Registrar terceros  
✔ Listar terceros
